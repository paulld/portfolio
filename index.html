About me

I am Paul Lugagne Delpon,
a former investment banker turned passionate web developer.
I made the move to do things I love!

I not only do my best to build great applications, but also enjoy brainstorming, planning, and implementing business ideas. I work hard to build web applications that are accessible, usable, functional, and beautiful.

Two of my favorite activities, coding and sports, have the same keys to success - fun and practice!

I enjoy learning new frameworks and languages, with a current preference for Meteor, Ruby on Rails, AngularJS, and Ember. I use the PostgreSQL and MongoDB databases extensively, and I spend significant time improving my skills in usability, accessibility (to people with disabilities), and Agile web development.

Selected work

Predikt

Predikt is a sports bet simulation app. Users can use in-game currency to practice their gambling skills, test strategies, interact with other users, and share betting tips.

Frameworks, Database & Others: Ruby on Rails, AngularJS, PostgreSQL, built-in JSON API following the json:api standard, built-in authentification process.

Languages: Ruby, CoffeeScript, HTML5 / Slim, CSS3 / Stylus, Responsive Design.

Chattr

Chattr is Chat application running on Meteor, part of an open-source course management system. It will allow creating archivable vs. temporary chatrooms, sharing links, code snippets, and files. It requires GitHub authentification.

Framework & Database: Meteor.js, MongoDB, GitHub login.

Languages: CoffeeScript, HTML5, Stylus / CSS3, Less, Responsive Design.

View Repo

ArtyPedia

Artypedia is a Wiki-like Art library project. It aims at gathering in one place all major artists, and referencing them to their complete list of artworks. It currently concentrates on modern art.

Framework: Ember.js, using the Ember-CLI executable.

Languagues: CoffeeScript, EmberScript, Emblem, Stylus.
